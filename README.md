# Projet-Lacosina

Lacosina est un projet que j'ai pu faire durant mes TP de R3.01 Développement Web.
Ce projet consiste en la création d'un site de cuisine. Les utiliseur peuvent ajouter, supprimer ou modifier des recettes celon leus droit (anonymes, membres ou admin).
Il m'a permis de développer mes compétences en PHP, Javascript et en développement web en générale. Mais aussi, l'utilisation d'une base de données MariaDB

## Mise en place 

Utilisation de Wampserveru 64 sur windows pour le serveur PHP ainsi que la base de données MariaDB en locale.
Utilisation de Bootstrap pour le CSS ainsi que Composer. J'ai aussi pu mettre en place des logs.
